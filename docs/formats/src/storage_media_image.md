# Storage media image formats

A storage media image format is used to store data from storage media devices
such as a hard disk, a floppy or optical disk like CD-ROM or DVD.

## Formats

* [Expert Witness Compression Format (EWF)](ewf.md)
* [Expert Witness Compression Format version 2 (EWF2)](ewf2.md)
* [Mac OS sparse bundle](sparsebundle.md)
* [Mac OS sparse image](sparseimage.md)
* [Parallels Disk Image (PDI)](pdi.md)
* [QEMU Copy-On-Write (QCOW)](qcow.md)
* [Universal Disk Image Format (UDIF)](udif.md)
* [Virtual Hard Disk (VHD)](vhd.md)
* [Virtual Hard Disk version 2 (VHDX)](vhdx.md)
* [VMWare Virtual Disk Format (VMDK)](vmdk.md)
